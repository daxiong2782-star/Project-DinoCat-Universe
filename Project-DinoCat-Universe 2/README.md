# 🦖 Project DinoCat Universe

Version: `0.1.0`  
Status: `Milestone 01 / Project Foundation`

Project DinoCat Universe 是一个 AI 原生原创 3D 动画 IP 工程。

核心主角：**龙宝 LongBao**，一只穿着米白色猫玩偶服的绿色小恐龙，为了完成成年礼，来到地球旅行，收集 365 枚彩虹徽章。

## 项目目标

打造一个可长期运营、可跨平台分发、可商业化授权的原创 AI 动画 IP。

## 内容方向

- 短视频
- AI 动画
- 连续短剧
- 旅行故事
- 治愈搞笑内容
- 表情包与文创
- 后续游戏、动画剧集、品牌联名

## 生产流水线

Idea → Research → Script → Storyboard → Image Prompt → Video Prompt → Voice → Editing → QA → Publish → Review → Knowledge Base

## 核心原则

任何内容都必须先遵守：

1. `AGENTS.md`
2. `bible/` 系列文档
3. `seasons/` 季度剧情
4. `daily/` 每日制作包

没有文档依据，不允许随意修改角色、世界观、画风和剧情规则。
