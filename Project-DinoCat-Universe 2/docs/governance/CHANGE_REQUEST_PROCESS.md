# Change Request Process

## Purpose
Protect IP consistency while allowing controlled iteration.

## Change Types

### Minor Change
Examples:
- title improvement
- subtitle correction
- platform copy adjustment

Can be changed directly.

### Major Change
Examples:
- character appearance
- world rule
- canon timeline
- new main character
- visual style

Must create a change request.

## Change Request Template

```md
# Change Request

## Requested Change

## Reason

## Affected Files

## Risk

## Decision
Approved / Rejected / Needs Revision
```

## Rule
No major change can override `AGENTS.md` without explicit approval.
