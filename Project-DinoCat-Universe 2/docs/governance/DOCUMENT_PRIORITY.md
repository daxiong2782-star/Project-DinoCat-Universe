# Document Priority

When documents conflict, follow this order:

1. `AGENTS.md`
2. `PROJECT_CHARTER.md`
3. `bible/universe/UNIVERSE_BIBLE.md`
4. `bible/characters/LONGBAO_CHARACTER_BIBLE.md`
5. `bible/visual/VISUAL_BIBLE.md`
6. `bible/prompts/PROMPT_BIBLE.md`
7. `bible/production/PRODUCTION_BIBLE.md`
8. Season documents
9. Daily episode documents
10. Marketing variations

Daily production can adapt platform copy, but cannot change canon, character design, or worldview.
