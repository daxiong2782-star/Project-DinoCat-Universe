# Daily Workflow

## Morning
1. Read `AGENTS.md`
2. Read canon documents
3. Read yesterday analytics
4. Create today's episode package
5. Run QA checklist

## Production
1. Generate images
2. Generate video clips
3. Generate voice
4. Edit and subtitle
5. Create cover
6. Export final video

## Publish
1. Publish to primary platform
2. Adapt copy for secondary platforms
3. Record publish time

## Evening
1. Record analytics
2. Collect comments
3. Update `codex/memory/MEMORY.md`
4. Create tomorrow suggestion
