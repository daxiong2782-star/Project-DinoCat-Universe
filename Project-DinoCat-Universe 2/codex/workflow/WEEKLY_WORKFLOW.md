# Weekly Workflow

## Weekly Goals
- Review 7 episodes
- Find strongest hook
- Find weakest retention point
- Update prompt rules
- Update content direction
- Plan next 7 episodes

## Required Report
Create `analytics/weekly/YYYY-WW_WEEKLY_REPORT.md`.

Report sections:
- best video
- worst video
- audience comments
- winning patterns
- failed patterns
- next week strategy
