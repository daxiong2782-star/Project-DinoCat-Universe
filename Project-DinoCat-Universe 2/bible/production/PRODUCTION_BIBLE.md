# Production Bible v0.1

## 每日制作流程

1. 读取 AGENTS.md
2. 读取 Universe Bible
3. 读取 LongBao Character Bible
4. 读取 Visual Bible
5. 读取 Prompt Bible
6. 生成当天剧本
7. 生成分镜
8. 生成图片 Prompt
9. 生成视频 Prompt
10. 生成配音、字幕、标题、文案
11. QA 检查
12. 发布
13. 复盘并写入 Analytics

## 发布前 QA

- 角色是否一致
- 场景是否清楚
- 前 3 秒是否有钩子
- 是否有文化元素
- 是否有小冲突
- 是否有温暖解决
- 是否符合家庭友好调性
