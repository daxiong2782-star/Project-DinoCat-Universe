# LongBao Master Assets v1

Official master asset structure for LongBao.

## Directory Structure

- `turnaround/` — official front, back, left, right, and 45-degree views
- `expressions/` — standard expression sheet and individual expression crops
- `actions/` — standard action sheet and individual action crops
- `props/` — growth journal, backpack, camera, logo/slogan references
- `color/` — official color palette reference
- `material/` — official material guide reference

## Usage Rule

All future images, video prompts, storyboards, model references, and merchandise references should use this folder as the master visual reference.
