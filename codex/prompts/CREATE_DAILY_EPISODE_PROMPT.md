# Create Daily Episode Prompt

Create a complete daily production package for `DayXXX`.

Required inputs:
- target country / city
- episode theme
- character growth point
- platform priority

Required output files:
- `daily/dayXXX/Script.md`
- `daily/dayXXX/Storyboard.md`
- `daily/dayXXX/ImagePrompt.md`
- `daily/dayXXX/VideoPrompt.md`
- `daily/dayXXX/Voice.md`
- `daily/dayXXX/Subtitle.md`
- `daily/dayXXX/Publish.md`
- `daily/dayXXX/Review.md`
- `daily/dayXXX/ProductionLog.md`

Rules:
- Follow `AGENTS.md`
- Keep LongBao visually consistent
- Keep story warm, funny, and all-age safe
- One episode should be 30–60 seconds
- End with emotional hook or next-day curiosity
