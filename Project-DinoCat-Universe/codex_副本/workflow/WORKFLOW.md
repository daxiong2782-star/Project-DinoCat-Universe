# Codex Workflow

## Daily Run

1. Read `AGENTS.md`
2. Read current Bible files
3. Read target `daily/dayXXX/`
4. Generate missing production files
5. Run QA Checklist
6. Prepare publish package
7. Ask for human approval
8. Update Review after data is available

## Never Do

- Do not redesign LongBao without approval
- Do not change canon casually
- Do not generate violent or hateful content
- Do not skip QA
