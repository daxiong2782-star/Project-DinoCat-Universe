# Day001 Script — 日本：第一次来到地球

## Logline

龙宝第一次来到东京，被自动售货机吓了一跳，却因此认识了一只流浪猫，并获得第一枚彩虹徽章。

## Duration

40–50 秒。

## Opening Hook

龙宝刚到东京，就对着一台自动售货机大喊："这是一只会吐饮料的机器人吗？"

## Story

龙宝拖着小行李箱来到东京街头，看见东京塔和发光的招牌。他发现一台自动售货机，好奇地按下按钮。汽水滚出来后，他兴奋地摇了摇，结果打开瞬间喷得到处都是。

路边一只小猫被喷湿了。龙宝连忙道歉，用自己的小毛巾帮小猫擦干。小猫没有生气，反而带他找到附近的神社小路。

夕阳下，小猫把一枚发光的小徽章推到龙宝面前。龙宝获得第一枚彩虹徽章。

## Growth Point

好奇会带来麻烦，但真诚道歉和补救会带来新的朋友。

## Ending Hook

龙宝翻开地图："下一站，会遇见谁呢？"
