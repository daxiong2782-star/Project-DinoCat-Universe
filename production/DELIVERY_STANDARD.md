# Delivery Standard

## Short Video
- Duration: 30–60 seconds
- Format: vertical 9:16
- Resolution: 1080x1920 minimum
- Subtitle: required
- Cover: required
- Audio: clean and non-distorted

## Daily Package
Every published episode must archive:
- final script
- storyboard
- prompts
- final publish copy
- analytics report
- production notes

## Naming
`DinoCat_DayXXX_Platform_Version.ext`

Example:
`DinoCat_Day001_Douyin_v1.mp4`
