# Production QA Checklist

## Canon
- [ ] LongBao is still a green baby dinosaur.
- [ ] LongBao wears the same beige cat onesie.
- [ ] No unauthorized character redesign.
- [ ] Story does not violate world rules.

## Visual
- [ ] 3D high-quality animated look.
- [ ] Warm lighting.
- [ ] Round, cute proportions.
- [ ] No horror, violence, or disturbing visuals.

## Story
- [ ] Clear hook in first 3 seconds.
- [ ] One simple conflict.
- [ ] Warm resolution.
- [ ] Growth point included.
- [ ] 30–60 seconds.

## Platform
- [ ] Title has curiosity.
- [ ] Cover is readable.
- [ ] Hashtags are relevant.
- [ ] Comment question included.

## Final Decision
- [ ] Approved
- [ ] Needs revision
- [ ] Rejected
