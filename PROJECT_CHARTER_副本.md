# Project Charter

## 项目名称

Project DinoCat Universe（恐龙猫宇宙）

## 项目目的

建立一个可持续生产内容、可跨平台运营、可商业化授权的原创 AI 动画 IP。

## 核心 IP

龙宝 LongBao：一只穿着猫玩偶服的小恐龙。

## 项目周期

2026–2036

## 第一年度目标

- 完成 365 天短视频内容体系
- 建立 IP Bible、Character Bible、Visual Bible、Prompt Bible
- 建立 Codex 自动化生产流程
- 完成第一季主线内容
- 形成可复用的角色、场景、动作和 Prompt 资产库

## 成功指标

- 365 个完整每日制作包
- 100 条以上可发布短视频
- 1 套可持续迭代的项目仓库
- 多平台账号矩阵启动
- 形成可商业化的 IP 资产
